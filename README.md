# Riddle Language
简体中文 | [English](./README_en.md)

轻便，灵活，现代化的编程语言。

## 介绍
Riddle Language 是一个内存安全的编程语音，用于取代 Rust 而出现。

Riddle 支持所有现代语言所支持的特性：
1. 闭包
2. λ 表达式
3. ....
